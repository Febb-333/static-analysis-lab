# Portofolio Eksperimen Reverse Engineering & Analisis Keamanan Siber

Dokumentasi ini dibangun sebagai bagian dari portofolio pribadi untuk memenuhi tugas mandiri dan kelompok pada mata kuliah Reverse Engineering. Seluruh aktivitas kompilasi, dekonstruksi biner, dan pelacakan kelayakan kode dicatat secara berkala untuk menunjukkan pola kerja yang konsisten.

## Disclaimer Tujuan Riset & Legalitas
Repo ini dibuat untuk tujuan edukasi mata kuliah Keamanan Siber — UNIVERSITAS AMIKOM YOGYAKARTA. Seluruh analisis kode dilakukan di lingkungan terisolasi (Virtual Machine). Berkas biner berbahaya tidak diunggah secara aktif ke repositori publik demi menjaga kepatuhan etika keamanan informasi.

## Struktur Utama Portofolio
- `/docs` : Berkas writeup komprehensif berformat Markdown.
- `/images` : Dokumentasi grafis orisinal disertai anotasi analitis.
